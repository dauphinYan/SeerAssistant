### 功能

- 针对赛尔号（Flash端）网络通信中数据包解密。

### 运行

#### 编译 `SocketHook.dll`

编译 `SocketHook.dll` ，在终端中运行

```
mingw32-make（根据实际情况修改）
```

> 编译前，请确保你已安装 MinGW-w64，并已配置环境变量

#### 运行`Injector.cpp`

运行`Injector.cpp`（主程序），注意修改`targetExePath`为游戏启动路径。
